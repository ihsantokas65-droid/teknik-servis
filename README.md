# Teknik Servis Web Sitesi (Next.js)

SEO uyumlu **kombi / klima / beyaz eşya teknik servis** sitesi için Next.js (App Router) başlangıç projesi.

## Kurulum

```bash
npm install
```

## Çalıştırma

```bash
npm run dev
```

## Site ayarları

`NEXT_PUBLIC_SITE_URL` ile canonical + sitemap URL’lerini ayarlayın:

```bash
# .env.local
NEXT_PUBLIC_SITE_URL=https://www.ornek-domain.com
```
